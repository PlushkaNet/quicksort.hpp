# quicksort.hpp
## One-header library with simple but fast sorting method
> **You can look into *quicksort_example.cpp* to see example of usage**
<br>

> C single header library quicksort.h can be found **[here](https://github.com/PlushkaNet/quicksort.h)**
## Avaliable methods:
- **`qs::quicksort`**
- **`qs::quicksort_s`**
- **`qs::is_sorted`**
- **`qs::reverse`**
- **`qs::is_sorted_reverse`**
<br>

This library uses its own namespace `qs::`
